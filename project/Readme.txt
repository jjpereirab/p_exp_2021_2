Z BOSON ANALYSIS (8 TeV)

The project was made through the same framework used for the Z' exercise. 

The modified files were: 

-TopAnalysis.C
-TopAnalysis.h
-Tophistograms.h
-HistPlotter.C

Most modifications are in TopAnalysis.C which contains the selection criteria for the leptonic channel in question.
